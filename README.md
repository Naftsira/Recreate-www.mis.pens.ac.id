<h1>Recreate My Campus Information System Blast with Authentication System</h1><br>
Express + TailwindCSS + MongoDB + Vercel<br>
<i><h5><--In Order to Web Design Assignment on First Semester--></h5></i><br>

This project is a replication of PENS System Information Blast designed as the campus information center as a campus information center to find faculty contacts, student information, and other campus-related information.

The web is built with Express and MongoDB with a focus on scalable architecture and fast deployment with Vercel.
The UI on site is built with TailwindCSS for fast and relieble design.
